**Pixel Panic Test** is a development and testing version of **Pixel Panic**.
This repository exists to experiment with features, test mechanics, and preview updates before they reach the main build.

⚠️ This repo may contain **unfinished or experimental features.**

---

# 🚗 Features

* **Endless 3D Mode** – Drive through procedurally generated roads and avoid hazards
* **Mini 2D Arcade Mode** – Retro pixel racing gameplay
* **Custom Map Builder** – Design and share your own tracks
* **Garage Upgrades** – Improve traction, armor, and nitro boosts
* **Pixel Art Graphics** – Classic pixel aesthetic in both 2D and 3D
* **Secret Cheat Codes** – Unlock hidden abilities and power-ups
* **Dynamic Music & Sounds** – Music adapts to gameplay intensity

---

# 🎮 Controls

| Key       | Action          |
| --------- | --------------- |
| **W**     | Accelerate      |
| **S**     | Brake / Reverse |
| **A**     | Steer Left      |
| **D**     | Steer Right     |
| **SHIFT** | Nitro Boost     |
| **F3**    | Debug menu      |
